# Student Management System (SQL)

A SQL database project to manage student records.

## Features
- Create Students database and table
- Insert student data
- View all students
- Update student marks
- Delete student records

## SQL Commands Used
- CREATE DATABASE
- CREATE TABLE
- INSERT INTO
- SELECT
- UPDATE
- DELETE

## How to Run
1. Open SQL Server Management Studio (SSMS)
2. Open StudentManagement.sql file
3. Execute the queries one by one

## Technologies Used
- Microsoft SQL Server
- SQL Server Management Studio (SSMS)
